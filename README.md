## 🚀 What I'm Building

- 🏢 **Enterprise Expense Agent & MCP Gateway** — AI expense assistant in Microsoft Teams on a multi-tenant .NET 10 MCP Gateway: OAuth 2.1 authorization server, Entra ID identity passthrough, 580+ tests, full Azure CI/CD *(BISTEC Global)*
- 🛠️ **AI IT-Support Platform for ServiceNow** — Copilot Studio agent connected to ServiceNow through a custom MCP gateway with 15+ tools, Cosmos DB multi-tenancy, and Azure Marketplace packaging *(BISTEC Global)*
- ✈️ **[Travel Plan Platform](https://travel-plan.live/)** — AI-powered travel marketplace for Sri Lanka: 13 Spring Boot microservices, saga-orchestrated bookings, Kafka-driven reviews, and a conversational agent that plans multi-day trips in under 2 minutes *(Final Year Project)*
- 📄 **DocMind** — production RAG document Q&A platform: LlamaIndex + Qdrant + GPT-4o-mini with cited answers, scale-to-zero on Azure Container Apps
- 🎓 **[QuizBank](https://quizbank.ai.bistecglobal.com)** — multi-tenant AI evaluation platform with GPT-4o question generation — .NET 9 Clean Architecture + Next.js 15
